#### How to Become a Contributor
We welcome contributions from the community to help improve RAGVue!
Before contributing, please read the following guidelines to ensure a smooth process.

To contribute:
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes with clear commit messages.
4. Push your changes to your forked repository.
5. Open a Pull Request against the main repository.
6. Describe your changes and the problem they solve in the PR description.
7. The maintainer will review your PR and may request changes.
8. Once approved, your changes will be merged into the main codebase.
